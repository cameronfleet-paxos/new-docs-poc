# Developer Guide Overview

Welcome to the Paxos Developer Guide! This documentation will help you integrate with the Paxos APIs to build powerful blockchain and digital asset solutions.

## What You Can Build

The Paxos APIs provide infrastructure to:

- **Accept stablecoin payments** - Integrate payment acceptance with Paxos-issued stablecoins
- **Move fiat funds** - Transfer fiat currency via wire or CUBIX
- **Facilitate settlements** - Handle pre-funded settlement workflows  
- **Mint, convert, and redeem** - Work with Paxos-issued stablecoins
- **Add crypto services** - Integrate cryptocurrency brokerage into your application

## Getting Started

Ready to start building? Here are the key steps:

1. **[Create a Developer Account](/getting-started/account)** - Sign up for sandbox access
2. **[Set up Authentication](/getting-started/authenticate)** - Configure API credentials
3. **[Explore the Sandbox](/getting-started/sandbox)** - Test your integration safely
4. **[Make Your First Request](/getting-started/request)** - Send your first API call
